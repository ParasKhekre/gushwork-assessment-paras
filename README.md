# Gushwork Assignment

Create a responsive web page using vanilla HTML, CSS, and JavaScript only (no
frameworks or libraries). The design specifications and requirements are detailed below.

## Project Structure

- `index.html` – Main page markup for the product landing experience.
- `styles.css` – Page styling, layout, and responsive design.
- `script.js` – Front-end interaction logic for carousel, sticky header, FAQ, modals, and more.
- `Asset/` – Image and icon assets used throughout the page.
